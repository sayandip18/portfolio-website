Google Summer of Code 2021

Worked as a student developer in the project PyData/Sparse under Python Software Foundation. Sparse aspires to replace scipy.sparse as the defacto sparse array implementation of the Python data ecosystem. Sparse is aiming to leverage the research done by the TACO team to perform Sparse array computations efficiently. TACO has a Python API, called pytaco, that makes the C++ code available in Python by wrapping it up in pybind11. Sparse aims to use the pytaco in a numpy.ndarray compatible API.

Currently a maintainer and contributor of the Sparse project.
